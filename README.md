# CpyTro (CPT) - Mobile Mining Cryptocurrency

CpyTro is a SHA-256 based cryptocurrency designed specifically for mobile mining with a maximum supply of 210,000,000 coins.

## Features

- **Mobile Mining**: Optimized for smartphones and tablets
- **Limited Supply**: Fixed maximum supply of 210M coins
- **SHA-256 Algorithm**: Proven security
- **Energy Efficient**: Battery-friendly mining
- **Easy to Use**: Simple mobile interface

## Installation

### Prerequisites
- Python 3.7 or higher
- 1GB free storage
- Internet connection

### Install from source

```bash
# Clone repository
git clone https://github.com/cpytro/cpytro-coin.git
cd cpytro-coin

# Install dependencies
pip install -r requirements.txt

# Run the node
python main.py
